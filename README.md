🎁 Вишлист (список желаний) – Multi‑Language Implementation
Тестовый репозиторий с реализацией консольного вишлиста на 7 языках программирования.
Приложение помогает вести список желаемых вещей, подарков или целей. Вы можете добавлять, редактировать, удалять, отмечать исполненные, фильтровать, искать, получать статистику и сохранять данные.

🎀 Что такое Вишлист?
Это личный органайзер для ваших желаний. Храните информацию о том, что вы хотите: название, описание, категория, приоритет (низкий/средний/высокий), цена (опционально), ссылка (опционально), статус (желаемое/исполненное). Приложение позволяет:

Добавлять желания с указанием всех полей.

Просматривать список с фильтрацией по статусу, категории, приоритету.

Искать по названию или описанию.

Редактировать и удалять записи.

Отмечать как исполненное.

Сортировать по приоритету, цене, дате добавления.

Получать статистику: общее количество, исполненные/неисполненные, средняя цена (если указана), распределение по категориям и приоритетам.

Сохранять и загружать данные в формате JSON.

Экспортировать и импортировать в CSV для работы с таблицами.

✨ Продвинутые возможности
Интерактивное меню с выбором действий.

Валидация полей: цена (≥0), приоритет (1–3), статус (0/1).

Автоматическая генерация ID и даты добавления.

Фильтрация по нескольким критериям.

Сортировка по возрастанию/убыванию.

Поддержка заметок (необязательное поле).

Сохранение в JSON и загрузка при старте.

Экспорт/импорт CSV (разделитель точка с запятой).

Цветной вывод (где поддерживается) для улучшения читаемости.
